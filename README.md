Hospital Readmissions Analysis
Tool: Microsoft Excel
Dataset: UCI Machine Learning Repository — Diabetes 130-US Hospitals 1999-2008
Records: 101,766
Project Overview
Analysis of diabetic patient encounter data from 130 US hospitals spanning 1999-2008 to identify factors associated with 30-day hospital readmissions. Includes full data cleaning documentation, ICD-9 diagnosis categorization, medication pattern analysis, and executive dashboard.
Key Findings

Overall 30-day readmission rate of 11.2%
Metformin stability associated with lowest readmission rates — 8.2% for increasing doses
Glipizide downward dose adjustments showed elevated risk at 15.2%
Insulin dose instability identified as marker of disease complexity not direct cause
